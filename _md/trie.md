## 説明
文字列を prefix でまとめた木。文字列挿入、prefix 検索が $O(|S|)$ でできる。

## できること
- `Trie<W, F> trie(f)` : 文字変換関数 `f` を使う Trie を作る
- `add(s, x)` : 文字列 $s$ を id `x` で挿入する
- `find(s)` : $s$ の終端ノード番号を返す。見つからなければ `-1`
- `find(cur, c)` : ノード `cur` から文字 `c` で遷移した先を返す。無ければ `-1`
- `id(cur)` : ノード `cur` に対応する id を返す。終端でなければ `-1`
- `idxs(cur)` : ノード `cur` に対応する id 一覧を返す。無ければ空
